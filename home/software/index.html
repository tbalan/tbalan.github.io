Software

frailtyEM is an R package that may be used for fitting shared frailty models. For a Breslow-type non-parametric baseline hazard, several frailty distributions are supported: gamma, inverse Gaussian, positive stable, PVF-family. Both clustered failures and recurrent events are supported, and prediction curves are easily obtained.

The stable version is on CRAN and development version on GitHub. For a more detailed presentation of models and methods, as well as an overview of the implemented estimation approach, I wrote a comprehensive vignette for the package.

The package is still under development, so any feedback is welcome.
